    Bots v0.1.0 Bots.RootSupervisor

    Данный модуль является основным RootSupervisor процессом. В зависимости от режима использования бота модуль RootSupervisor запускает процесс BotsSupervisor для работы в режиме active (long-polling) и процесс WebSupervisor для работы в режиме passive (web-сервера).

    Summary

    Functions

    init()

    Метод init инициалиизирует процесс запуска Supervisor модулей. Спецификация ботов может быть представлена в виде переменной bots_spec. Эта переменная должна включать в себя все необходимые параметры (наименование бота, режим работы, настройки и тд). Данная переменная разбирается на части и из этих частей генерируется список ботов для запуска
